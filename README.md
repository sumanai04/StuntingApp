# StuntingApp
This web app is created for Desa Panyingkiran community website which is based on blogspot
<br>https://www.desa-panyingkiran-majalengka-bisa.id

The stunting calculator is made using data provided by the goverment as stated in Permenkes No. 2 Tahun 2020
<br>https://peraturan.bpk.go.id/Details/152505/permenkes-no-2-tahun-2020

The app's purpose is to check whether or not your child has stunted growth based off the age and body height that you've provided from the most accessible device within your surroundings.
The app is not meant to give a final diagnosis; that decision should always be given from a medical expert.
